# Ruffier-test
About : Tes ruffier membantu pengguna menilai keadaan sistem kardiovaskular tubuh mereka. Ini dirancang untuk memantau kinerja jantung selama aktivitas dan mengevaluasi efektivitas kinerja tersebut

| Name           | Content |
|----------------|---------|
| My_app.py      | File utama yang menjalankan aplikasi dan mengatur ScreenManager |
| instr.py       | File berisi teks instruksi untuk pengguna |
| second_win.py  | Halaman untuk **timer** (penghitungan waktu selama teks Ruffier) |
| final_win.py   | Halaman untuk **penghitungan hasil** dan analisis Heart Check / Ruffier index |
